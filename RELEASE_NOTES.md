# 電脳秘書ロキ 0.9.3-alpha

公開用クリーン版の最初のスナップショットです。

## 含まれるもの

- Rokidグラス用HUD/Geminiクライアント
- Androidスマホ用カレンダー・通知メール・音声・Health Connectブリッジ
- Bluetoothキーボード、スマホ入力、音声入力
- 表情付きマスコット、回答読み上げ、頭部ジェスチャー表示
- 現在地・天気・気温表示、Zoom/Wi-Fiショートカット
- 新しい「電脳秘書ロキ」アイコン
- スマホ–グラス間の共有トークン認証

## 0.9.2-alpha の変更

- 「明日の天気」「明後日の天気」を予定検索より優先し、Open-Meteoの予報へ直接振り分け
- 回答の読み上げ完了から待機へ移ると、入力欄・回答欄・状態文を自動消去
- マスコットと日時・歩数・天気を最上段、操作ボタンをその直下へ再配置
- 姿勢センサーの起動直後を除外し、平滑化後に正面を校正することで常時点灯を抑制
- 正面復帰後は約1秒でHUDを消灯し、誤判定時も8秒で再校正

## 0.9.3-alpha の変更

- 頭部ジェスチャーの起動しきい値を少し上げ、意図しない点灯を抑制
- マスコット・情報欄の直下へボタン列を詰め、会話欄も上へ拡大
- Zoomカメラ起動ボタンの表示名を `CAM` に変更

## 注意

- アルファ版であり、一般端末での再現性は未保証です。
- Gemini APIキーは利用者が用意してください。課金状態にかかわらず `429` / `503` は起こり得ます。
- ローカル連携はトークン認証付きHTTPです。信頼できるLANまたは個人テザリングでのみ利用してください。
- メールは通知アクセス、予定はAndroidカレンダー、歩数はHealth Connect、天気は概算位置とOpen-Meteoから取得します。
- APIキー、個人用カスタム指示、会話ログ、予定、メール、端末IDは含めていません。
- 公開スナップショットはソース中心で、一括ビルド環境と署名済みAPKは未提供です。
