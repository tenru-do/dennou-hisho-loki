# Assets

The glasses app includes a monochrome mascot sheet:

```text
glass-app/assets/mascot_sheet.png
```

This asset is part of the personal prototype. Before public release, decide one of the following:

1. Keep it only if you are comfortable publishing it and can describe its source/license.
2. Replace it with a newly commissioned or clearly licensed asset.
3. Remove it and keep only the fallback code-drawn mascot.

For a public open-source repository, option 2 is the cleanest long-term path.
