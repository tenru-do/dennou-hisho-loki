# Roadmap

Possible next cleanup tasks before a public release:

- Convert the prototype into standard Gradle Android projects.
- Replace `com.example.*` package names with stable public package names.
- Add reproducible build scripts.
- Add screenshots or a short demo video with personal data removed.
- Add a real license file.
- Add a clear configuration flow for Gemini models and API keys.
- Add clearer model/quota guidance for Gemini 429 mitigation.
- Tune retry/cooldown intervals for practical daily use.
- Improve voice input reliability.
- Improve mobile-network/tethering behavior outside home Wi-Fi.
- Split experimental proactive AI code behind a safer feature flag.
- Add tests for calendar date/range parsing.
- Add issue templates for device/firmware compatibility reports.
