# セキュリティ

## 公開版の対策

- Gemini APIキーやOAuth資格情報を同梱しない
- Androidバックアップを無効化
- スマホのLANブリッジを32文字以上の共有トークンで認証
- トークン比較はタイミング差を抑える実装
- 個人用IP、端末シリアル、通知トピックを削除
- ビルド成果物、署名鍵、端末ダンプ、ログを `.gitignore` で除外

## 残るリスク

- ローカルブリッジはHTTPであり、通信内容は暗号化されません。
- トークンを画面共有、ログ、スクリーンショットへ写さないでください。
- APIキーが漏れた場合はGoogle AI Studioで直ちに無効化してください。
- メール通知アクセスは通知本文を読める強い権限です。不要なら許可しないでください。
- 本アプリは侵入試験や第三者監査を受けていません。

脆弱性を見つけた場合、予定・メール・APIキーなどを公開Issueへ貼らないでください。再現に秘密情報が必要な場合は、まず情報を伏せたIssueで連絡してください。
